# Modbus глазами чайника

## Описание
Случайно узнал, что существуют протокол modbus и язык C#.
Специально решил на всем этом поучиться.
Чайник был выбран как наглядный бытовой прибор.

## Дальнейший план.
1. Сделать интерфейсы для модбас-клиентов и модбас-протоколов.
2. Расширить функционал чайника(установка максимальной температуры).
3. Подключить еще одно устройство(кондиционер? увлажнитель воздуха?).
4. Вынести логику "инициализации" устройств в отдельный класс.
5. Добавить другие модбасы: аски, рту.

## Зачем.
1. Поработать с ООП.
2. Попрактиковать паттерны.
3. Познакомиться с "низким" уровнем.
